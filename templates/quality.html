Quality: About This Data Set

The data set presented in this site is about dog bite injuries. The data was provided by a legal firm, and is a collection of injury statistics associated with breeds of dogs that span from 1982 to 2012. For the most part, the data is good quality, although there are some missing datum.

Is the data complete?

The data was not entirely complete. Some categories of injuries associated with a certain breed were missing or blank. This did not raise much concern, as there were not many columns that had missing data. To account for this, I only plotted non-zero statistics for each breed of dog. After all, there were also a number of breeds that had zero for an injury statistic, so this would cause plots to be even more skewed.

Is the data coherent?

The data is pretty coherent in my opinion. I am not an expert in animals, but the data does seem to match my expectations, in that there would be breeds of dogs that seem to be more aggressive than others. This is evident in the fact that certain breeds, like the pitbull, have more counts of attacks than others. Due to the fact that there are many breeds of dogs, it makes sense that this distribution is also skewed.

In addition, before plotting, I had to clean up some data. As the data was stored via a JSON file instructed by the assignment, all values were strings. Thus, I had to convert numerical strings to actual integers. Also, as mentioned before, I removed all rows with nonzero statistics. Furthermore, there was a breed called "Total", which did not make sense for the purpose of plotting, so I removed that row as well before plotting.

Is the data correct?

The data seems correct. I made a Google search to research common dog breeds that cause injuries, and the statistics found in my research generally match the data presented in this Byte. You can read about it here.
